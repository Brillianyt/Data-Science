# SQL 练习：报告 2019 年春季才售出的产品

## 表结构

### 表：Product
| Column Name  | Type    | 备注         |
|--------------|---------|--------------|
| product_id   | int     | 主键         |
| product_name | varchar | 产品名称     |
| unit_price   | int     | 单价         |

### 表：Sales
| Column Name | Type    | 备注                   |
|-------------|---------|------------------------|
| seller_id   | int     | 销售员 ID              |
| product_id  | int     | 外键，关联 Product 表  |
| buyer_id    | int     | 买家 ID                |
| sale_date   | date    | 销售日期               |
| quantity    | int     | 数量                   |
| price       | int     | 售价                   |

## 需求
编写一个解决方案，报告仅在 **2019 年春季**（2019-01-01 至 2019-03-31，包含首尾日期）有销售记录的产品。这些产品在该时间段内必须有销售，且在其他任何时间段均无销售记录。

## 解决方案
sql

    SELECT

    p.product_id,

    p.product_name

    FROM Product p

    JOIN Sales s ON p.product_id = s.product_id

    GROUP BY p.product_id, p.product_name

    HAVING COUNT(s.sale_date BETWEEN '2019-01-01' AND '2019-03-31' OR NULL) = COUNT(*); #NULL用来表示不被计数，COUNT(*)表明除了这个时间段以外没有销售记录
